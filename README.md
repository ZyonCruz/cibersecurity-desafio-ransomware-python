# cibersecurity-desafio-ransomware-python
# Desafio: Implementação de Ransomware (Estudo em Python)

Este repositório contém a resolução do desafio da DIO sobre a implementação de scripts de criptografia para fins educacionais.

## 🛠️ Tecnologias
- Python 3
- Biblioteca `pyaes`
- Ambiente: Kali Linux

## 🚀 Como funciona
1. **encrypter.py**: Criptografa o arquivo `teste.txt` usando AES-CTR, gerando o arquivo `.ransomwaretroll`.
2. **decrypter.py**: Reverte o processo, recuperando o arquivo original.
